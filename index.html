Projetos

Estação Meteorológica Automatizada

Sistema desenvolvido para Raspberry Pi, integrando monitoramento ambiental, automação, captura inteligente de imagens e comunicação via bot Telegram para envio de alertas automáticos ou através de requisições manuais, contando ainda com gestão de usuários. O projeto utiliza sensor BME280, câmera Raspberry Pi, banco de dados PostgreSQL e rotinas em Node.js e Python para fornecer dados ambientais em tempo real, imagens processadas e relatórios automatizados sincronizados com servidor na núvem (InfluxDB e Grafana).

Entre as funcionalidades estão: leitura contínua de temperatura, umidade e pressão; geração de timelapses noturnos; captura de imagens com ajuste automático de exposição via IA; monitoramento de status do sistema; dashboards no Grafana; e gerenciamento de usuários diretamente pelo bot.
